### 5to Avance
**Presentación de Avance del Sistema utilizando MVC**

#### Resumen del Progreso:
Se ha realizado una  migración del proyecto, el cual inicialmente utilizaba únicamente JavaScript, HTML, Java, JSP y CSS. Ahora el proyecto emplea:

- **Spring Boot** como framework de Java para el backend.
- **React** como framework de JavaScript para el frontend.

#### Detalles de la Implementación:
1. **Modularicion**
    - Se moduralizo las paginas a componentes de react, con esto se logra a que los componentes sean reutilizables

2. **Controlador para Autenticación de Login:**
    - Se ha implementado un controlador dedicado a la autenticación de usuarios.
    - Este controlador verifica los usuarios existentes en un archivo JSON.


3. **Registro de Usuarios:**
    - Se ha implementado la funcionalidad de registro de usuarios.
    - Los nuevos usuarios registrados son agregados al archivo JSON para su autenticación futura.
